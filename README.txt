Lancement de l'application 


1. Installer Anaconda 
2. Ouvrir Anaconda Prompt 
3. executer : python app.py (installer les packs nécessaires : pip install -r "nom du pack")
4. se connecter à http://127.0.0.1:5000/ dans un nvaigateur (ou l'adresse indiquée)
